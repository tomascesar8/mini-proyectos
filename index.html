En esta página web he incluido algunos mini proyectos, realizados durante mis primeros pasos como desarrollador.

Si bien son simples, cada uno es una muestra de mi capacidad para resolver problemas y encontrar soluciones creativas en iniciativas de cualquier tamaño.

Ecommerce

Este proyecto es un ecommerce que cuenta con un sistema de login para los usuarios, un carrito de compras para agregar y quitar elementos, y una vista detallada para cada producto.

Se utilizaron tecnologías web como HTML, CSS, Bootstrap y JavaScript. La información de la sesión de usuario y los productos agregados al carrito se almacenan en el LocalStorage del navegador.

CRUD API Simpsons

Esta página web cuenta con un sistema CRUD permite visualizar y añadir personajes de la famosa serie de televisión Los Simpson.

Se utilizan tecnologías como HTML, CSS, Bootstrap y JavaScript, así como las funcionalidades básicas de Node.js necesarias para aplicar "npm json-server".

Lista de tareas

Crea una lista de tareas que permite agregar y eliminar tareas de forma dinámica en una página web.

Vaca en corral

Una vaquita que puede ser movida por el usuario utilizando las teclas de flechas. El movimiento se limita a un corral, delimitado por un rectángulo en la pantalla.
Se utiliza JavaScript para detectar el evento de pulsación de teclas, generando un cambio en las coordenadas de la vaca.

Qué día es hoy?

Reloj que muestra la fecha actual, junto con las horas, minutos y segundos en un formato de 12 horas con AM o PM.

Clon de Google

Réplica del buscador de Google en español

Cronómetro

Cronómetro con función para iniciar, pausar y detener el tiempo.

Temporizador

Temporizador que permite ingresar horas, minutos y segundos, y permite iniciar, pausar y detener la cuenta regresiva.

Tirar dado

Simula el lanzamiento de un dado de seis caras.
