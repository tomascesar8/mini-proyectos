En esta web he incluído proyectos simples realizados durante mis primeros contactos con JavaScript.

Fueron realizados con fin de practicar conceptos básicoss y escenciales de JavaScript.

Ecommerce

Ecommerce que cuenta con un sistema de login para los usuarios, un carrito de compras y una vista detallada para cada producto.

Desarrolado con HTML, CSS, Bootstrap y JavaScript. En cuanto al manejo de información para login y carrito de compras se utiliza en LocalStorage.

CRUD API Simpsons

Esta página permite visualizar y añadir personajes de la famosa serie de televisión Los Simpson.

Desarrolado con HTML, CSS, Bootstrap y JavaScript. En cuanto al manejo de información para login y carrito de compras se utiliza en LocalStorage.

En entorno local se utiliza el paquete "json-server" de Node.js para crear y consumir una base de datos a nivel local.

Al deployar el proyecto, por cuestiones prácticas e ilustrativas se utiliza LocalStorage para el manejo de información.

Lista de tareas

Lista de tareas que permite agregar y eliminar tareas.

Vaca en corral

Pasea a la vaca dentro del corral utilizando las teclas de flechas
El movimiento se limita a un corral, delimitado por un rectángulo en la pantalla.
Se utilizan event listeners de JavaScript para detectar el evento de pulsación de teclas, generando un cambio en las coordenadas de la vaca.

Fecha y hora

Reloj digital que muestra fecha y hora (horas, minutos y segundos) en un formato de 12 horas con AM o PM.

Clon de Google

Se clona el diseño y la interfaz del famoso motor de búsqueda Google en español.

Cronómetro

Cronómetro con función para iniciar, pausar y detener el tiempo.

Temporizador

Temporizador interactivo que facilita la cuenta regresiva en formato de horas, minutos y segundos, con funciones para iniciar, pausar y detener el conteo.

Lanzar dado

Simula el lanzamiento de un dado de seis caras.
